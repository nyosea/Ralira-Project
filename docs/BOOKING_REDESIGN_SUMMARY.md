# Redesign Form Booking - Simple & Clean

## Perubahan Desain
Mengubah form booking dari desain "norak dan gede" menjadi simple, clean, dan user-friendly.

## Desain Baru - Simple & Clean

### 1. **Color Scheme**
- **Background**: Light gray (#f8f9fa) - lebih soft dan eye-friendly
- **Text**: Dark gray (#2c3e50) - lebih mudah dibaca
- **Primary**: Blue (#007bff) - professional dan calming
- **White**: Clean white background untuk form

### 2. **Typography**
- **Font**: 'Segoe UI' - modern dan clean
- **Header**: 28px, weight 600 - tidak terlalu besar
- **Labels**: 14px, weight 500 - readable
- **Form elements**: 14px - comfortable size

### 3. **Layout & Spacing**
- **Container**: Max-width 700px (dari 900px) - lebih compact
- **Form padding**: 30px (dari 40px) - lebih breathable
- **Section margin**: 30px (dari 35px) - balanced spacing
- **Grid layout**: Responsive 2-column - efficient space usage

### 4. **Form Elements**
- **Input height**: 38px (dari 46px) - standard size
- **Border radius**: 6px (dari 10px) - subtle rounded
- **Border**: 1px solid (dari 2px) - cleaner look
- **Focus states**: Blue outline dengan shadow - clear feedback

### 5. **Header Design**
- **Removed**: Gradient background, decorative elements
- **Simple**: Centered text dengan subtle styling
- **Clean**: Hanya judul dan subtitle tanpa icon besar

### 6. **Modal Design**
- **Simplified**: Clean header dengan proper close button
- **Organized**: Proper header, body, footer structure
- **Reduced**: Less padding dan simpler styling

### 7. **Visual Elements**
- **Removed**: Complex gradients, shadows, animations
- **Simplified**: Subtle hover effects
- **Clean**: Minimal decorative elements
- **Professional**: Business-like appearance

### 8. **Responsive Design**
- **Mobile-first**: Proper mobile breakpoints
- **Grid**: Responsive 1-column pada mobile
- **Readable**: Proper text sizes pada mobile
