# Ralira-Project
A web-based information system designed to support the operational activities of Biro Psikologi Ralira.
This system helps manage client registration, consultation scheduling, psychological test results, and automated communication between admin, psychologists, and clients.

The platform focuses on improving service efficiency, reducing manual administration, and providing a better digital experience for clients.
